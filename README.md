## Hướng dẫn sử dụng.

1. Cài đặt Hugo

Cài đặt [ở đây](https://gohugo.io/getting-started/installing/)

2. Sử dụng mã nguồn
```bash
git clone https://github.com/vinalog/vinalog.github.io.git

cd vinalog.github.io

cd themes

git clone https://github.com/CaiJimmy/hugo-theme-stack.git

cd ..

hugo server -D

```
